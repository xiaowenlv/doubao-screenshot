# 豆包截图助手

一个极简的 Android 悬浮球工具，一键截图并直接发送给豆包 AI 识别。

## 功能

- 📸 **悬浮球**：常驻屏幕右侧，任意 App 内可用，可拖动位置
- 👆 **三指三击**：在任意界面快速敲击屏幕三下触发截图（需开启无障碍服务）
- 🤖 **直达豆包**：截图后自动通过系统分享接口发送给豆包 App

## 下载 APK

在 [Actions](../../actions) 页面找到最新一次构建，下载 `doubao-screenshot-debug` 压缩包，解压得到 APK。

## 安装与使用

### 第一步：安装 APK
手机允许「安装未知来源应用」，然后安装 APK。

### 第二步：授权
打开 App，按提示完成两项授权：
1. **悬浮窗权限**（必须）— 用于显示悬浮球
2. **无障碍服务**（可选）— 用于三指三击手势

### 第三步：启动
点击「启动悬浮球」，之后可以关闭 App 页面。

### 使用方式
| 方式 | 操作 |
|------|------|
| 悬浮球 | 点击屏幕右侧悬浮球（读屏模式下双击） |
| 三指三击 | 三根手指快速敲击屏幕三下 |

截图后豆包会自动打开，图片已在输入框中，直接输入问题发送即可。

## 权限说明

| 权限 | 用途 |
|------|------|
| 悬浮窗 | 显示悬浮球按钮 |
| 前台服务 | 保持服务在后台运行 |
| MediaProjection | 截取屏幕内容（首次需用户确认） |
| 无障碍服务 | 监听三指三击手势（不读取任何屏幕内容） |

## 构建

```bash
./gradlew assembleDebug
```

APK 输出路径：`app/build/outputs/apk/debug/app-debug.apk`
